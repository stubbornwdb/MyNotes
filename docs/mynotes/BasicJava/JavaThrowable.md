# Java异常处理

### 1.异常体系结构

分类：

- Error   JVM无法解决的严重问题，不需编写针对性代码进行处理；

  - StackOverflowError   递归调用可导致

  - OOM    Integer[] arr = new Integer[1024 * 1024 * 1024];

  

- Execption 其他应编程错误或偶然外在因素导致一般性问题，可以使用针对性的代码进行处理

  - 空指针访问
  - 试图读取不存在的文件
  - 网络连接中断
  - 数组下标与为记为

解决方法：

- 遇到错误终止程序进行
- 编码时考虑到错误的检测、消息提示及错误的处理（捕获错误的最理想时间时间编译期间，但有的错误在运行时才会发生）

异常处理机制

- try-catch-finally 
- throws
- 手动抛出异常 throw
- 用户自定义异常类
